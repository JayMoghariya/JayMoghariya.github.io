Probability and Statistics

  • Important Formulas

    • Basics of Probability

      1. $P(A^{c}) = 1 - P(A)$
      2. $P(A \cup B) = P(A) + P(B) - P(A \cap B)$
      3. $P(A \cap B) = P(A)*P(B/A)$
      4. $P(A/A) = 1$
      5. $if\space A,B,C \space are\space mutually \space independent \space events \space \to \space P(A \cap B \cap C) = P(A)*P(B)*P(C) $
      6. $if \space A \cap B = \phi, \space then \space P(B/A) = 0$
      7. $ P(B \cap C / A) = P(B/ (A \cap C)) * P(C/A) $
      8. $P(A_1 \cap A_2 \space \cap ... A_n) = P(A_1)*P(A_2/A_1)*P(A_3/A_2 \cap A_1)*...P(A_n/A_1 \cap A_2 \space \cap ... A_{n-1})$
      9. $P(B) = \sum_{i}^{}P(B/A_i)*P(A_i)$
      10. $Bayes' \space Theorem: \space P(A_i/B) = \frac{P(A_i)*P(B/A_i)}{\sum_{j}^{}P(A_j)*P(B/A_j)}$

    • Cumulative Distribution Function [F(x)]

      1. $F(x) = P(X \leq x), \space where \space P(X) \to Probability \space Mass \space Function \space (PMF)$
      2. $F(x) = \int_{-\infty}^{x}f(t)dt, \space f(t) \to Probability \space Density \space Function \space (PDF)$

    • Mean and Variance

      1. $E(x) = \int_{-\infty}^{\infty}x*f(x)\space dx \space\space \textbf{or} \space\space E(x) = \sum_{i}^{}x_i * f(x_i)$
      2. $E(ax+b) = a*E(x) + b, \space where \space a,b \to constants$
      3. $if \space P(x\ge 0) = 1 \space \& \space E(x) \space exists \space \Rightarrow \space E(x)\ge 0$
      4. $E(g(x))=\sum_{i}^{}g(x_i)P(X=x_i) \space \textbf{or} \space \int_{-\infty }^{\infty} g(x)*f(x)dx$
      5. $E(x) = \int_{0}^{\infty}(1-F(x))dx - \int_{-\infty}^{0}F(x)dx, \space where \space F(x) \to Cumulative \space Distribution \space Function$
      6. $E(x)= \sum_{k=0}^{\infty}k*P(x=k) = \sum_{k=0}^{\infty}P(x>k)$
      7. $Var(x) \ge 0$
      8. $Var(x) = E[(x - \mu)^2] = E(x^2) - E(x)^2$
      9. $Var(ax+b) = a^2*Var(x)$
      10. $Var(x+y) = Var(x) + Var(y) + 2Cov(x,y)$
      11. $Cov(x,y) = E[(x-\mu_x)(y-\mu_y)] = E(xy) - E(x)E(y)$
      12. $Cov(x,x) = Var(x)$
      13. $Cov(x,y+z) = Cov(x,y) + Cov(x,z)$
      14. $Cov(ax+b,cy+d) = ac*Cov(x,y)$
      15. $Cov(x,y) \le \sqrt{Var(x)*Var(y)}$
      16. $Corr(x,y) = \frac{Cov(x,y)}{\sqrt{Var(x)*Var(y)}}$
      17. $Corr(x,y) \in [-1,1]$
      18. $Corr(x,y) = 1 \space \Rightarrow \space y = ax+b, \space where \space a>0$
      19. $Corr(x,y) = -1 \space \Rightarrow \space y = ax+b, \space where \space a<0$
      20. $Corr(x,y) = 0 \space \Rightarrow \space x,y \space are \space independent$
      21. $Var(x+y) = Var(x) + Var(y) + 2\sigma_x\sigma_yCorr(x,y)$
      22. $Var(x-y) = Var(x) + Var(y) - 2\sigma_x\sigma_yCorr(x,y)$
      23. $Var(ax+by) = a^2Var(x) + b^2Var(y) + 2ab\sigma_x\sigma_yCorr(x,y)$
      24. $\textbf{z-score :} \frac{x-\mu}{\sigma}$, z-score shows how many standard deviations the data point is from the mean.

    • Inequalities:

      1. $Markov \space Inequality: \space P(x\ge \epsilon) \le \frac{E(x)}{\epsilon}$
      2. $Chebyshev \space Inequality: \space P(| x- \mu| \gt \epsilon) \le \frac{\sigma^{2}}{\epsilon^{2}}, \space where \space \mu,\sigma^2 \to mean,variance$
      3. $Cauchy-Schwarz \space Inequality: \space E(XY)^2 \le E(X^2)E(Y^2)$
      4. $Jensen's \space Inequality: \space E(g(x)) \ge g(E(x)), \space where \space g(x) \to convex \space function$
      5. $Holder's \space Inequality: \space E(XY) \le (E(X^p))^{1/p}(E(Y^q))^{1/q}, \space where \space p,q \to conjugate \space exponents$
      6. $Minkowski's \space Inequality: \space (E(X+Y)^p)^{1/p} \le (E(X^p))^{1/p} + (E(Y^p))^{1/p}, \space where \space p \ge 1$

    • Central Limit Theorem

      1. In simpler terms, the Central Limit Theorem suggests that if you have a large enough sample size from a population with any distribution (as long as the sample is drawn randomly and the observations are independent), the distribution of the sample means will be approximately normal, even if the original population is not normally distributed.
      2. The Central Limit Theorem is often applied in hypothesis testing, confidence interval estimation, and statistical inference.
      3. As sampling size increase sample mean tends to normal distribution highly likely.

Machine Learning

  • Hypothesis Testing

    1. The null hypothesis typically represents the status quo or the hypothesis of no effect or no difference, while the alternative hypothesis represents the hypothesis you want to test or the claim you are interested in.
    2. In p-test, smaller p-values (typically less than 0.05) represents the strong evidence against null hypothesis. But p-value greater than 0.05 is insufficient evidence for rejecting null hypothesis.
    3. The p-value represents the probability of observing a sample mean as extreme as or more extreme than the one obtained, assuming the null hypothesis is true.
    4. White noise is a noisy data which can not be interrelated. In mathematical terms:
      • It has constant mean
      • It has constant variance
      • It has no auto correlation (we can use tree regressors for these)
    5. The t-test calculates a t-statistic, measures the difference between the means of the two groups relative to the variability within each group.

  • Linear Regression

    • Assumptions of Linear Regression
      1. Linear relationship between dependent and independent variables
      2. Independent variables are not correlated
      3. Residuals are normally distributed
      4. Residuals have constant variance
      5. Residuals are independent of each other
